# Riddle Game Mobile

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/emanuelikanani-glitch/riddle-game-mobile.git
   cd riddle-game-mobile
   ```

2. **Install dependencies**:
   Ensure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Run the application**:
   ```bash
   npm start
   ```

## QR Code Distribution Guide

To distribute your riddle game using QR codes, follow these steps:

1. **Generate a QR Code**:
   Use a QR code generator like [QRCode Monkey](https://www.qrcode-monkey.com/) to create a QR code linking to your app.

2. **Print the QR Code**:
   Include the QR code on flyers or posters to reach your audience.

3. **Share digitally**:
   Include the QR code in digital communications such as emails, social media posts, or on websites.

4. **Test the QR Code**:
   Ensure the QR code correctly links to your application before wide distribution.

5. **Feedback**:
   Encourage users to provide feedback on their experience with the game via the app or a dedicated feedback form.
